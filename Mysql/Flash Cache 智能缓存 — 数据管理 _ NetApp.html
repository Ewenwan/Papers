    NetApp Flash Cache

    • 概述
    • 技术规格
    • 联系销售部门
    • 4008 1818 11

    要点

    • 适用于包括数据库、电子邮件、文件服务在内的随机读取密集型工作负载。
    • 与硬盘驱动器相比，延迟时间降低 10 倍甚至更多。
    • 提高 I/O 吞吐量并消除性能瓶颈。
    • 将 SATA 驱动器与 Flash Cache 配合使用来处理重要的工作负载，从而降低成本。
    • 使用数量更少、容量更大的硬盘驱动器，可节省电力、散热成本和机架空间。

    使用 NetApp Flash Cache 智能缓存和容量优化驱动器提高存储性能和效率。

    通过添加 NetApp® Flash Cache（连接到控制器的 PCIe 智能缓存）来优化您的存储性能，而不是添加更多硬盘驱动器 (Hard Disk Drive, HDD) 或固态驱动器 (Solid State Drive, SSD)。

    Flash Cache 可智能缓存最近读取过的用户数据或 NetApp 元数据，从而加快数据访问。无需进行设置或者持续管理，还可以调整操作。Flash Cache 可与所有 NetApp 存储协议和软件一起使用，从而帮助您：

    • 将 I/O 吞吐量提高多达 75%。[1]
    • 在不影响性能的情况下，将磁盘驱动器使用量减少多达 75%。[2]
    • 在不添加磁盘驱动器的情况下，将电子邮件用户增加多达 67%。[3]

    智能缓存和 NetApp 存储效率技术支持虚拟存储层。虚拟存储层可将热数据实时提升到性能存储，而无需移动数据。

    详细了解 Flash Cache 和 Flash Cache 2 智能缓存。

    使用预测性缓存统计信息功能（NetApp® Data ONTAP® 存储操作系统的一项功能）模拟添加 Flash Cache 的结果。

    [1]请参阅 WP-7082《在联机事务处理中使用 NetApp Flash Cache (PAM II)》。
    [2]请参阅 TR-3799《Flash Cache 在文件服务工作负载中的应用》。
    [3]请参阅 TR-3867《在 Exchange 2010 中使用 Flash Cache》。
